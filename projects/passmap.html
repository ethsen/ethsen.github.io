Pass Probability Estimation (PassMap-style)

TL;DR
Predict a spatial heatmap for completion likelihood and (optionally) a destination distribution from a state tensor built from broadcast-derived freeze frames.
Code PDF (optional) Back to projects

1. Problem

Given a game state \(s\) (players/ball encoded on a grid), estimate the probability a pass to cell \(g\) is completed: \[ P(\text{complete}\mid s, g). \] Optionally, also estimate a destination distribution \(P(g\mid s)\) via a softmax head.

2. Approach

We represent the state as a multi-channel tensor \(X \in \mathbb{R}^{C\times H\times W}\) and train a CNN to output a heatmap over grid cells. A common pattern is two heads: one sigmoid heatmap for completion and one softmax heatmap for destination. 

# Sketch (conceptual)
X: (C,H,W)  ---> CNN backbone ---> shared features F
                                  |             |
                                head 1        head 2
                               sigmoid        softmax
                            P(complete|s,g)    P(g|s)
